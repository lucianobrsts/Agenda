
## Agenda Java
> O projeto Agenda foi idealizado para conhecer e firmar conceitos sobre a linguagem Java. Faz uso de tecnologias essênciais para automação básica de processos e alguns frameworks de ponta do mercado DevJava . 

#### Tecnologias utilizadas:

- Windows 10
- Eclispe Oxygen
- Java 7
- JavaScript
- Ajax
- Git 2.19
- Junit 4.12
- JSF 2.2
- Primefaces 4.0
- IReport 5.6
- Jasper Report 5.6
- MySQL
- Ireport 5.6

#### Funcionalidade:

> O projeto tem como funcionalidade principal constrolar o fluxo de cadastro de contatos pessoais e profissionais de seus usuários. Utiliza uma interface amigável e conhecida. Possui um CRUD da entidade Contatos e outro da entidade Usuário. A renderização de todas as views envolve JSF e Primefaces(onde o back executa Ajax e javaScritp). Por convenção ainda do ambiente acadêmica optou-se pelo teste unitário para verificar a execução de todos os métodos afins. Como ferramenta de design de relatórios usou-se a ferramenta IReport e seu framework JasperReport no processo de desenho e geração de relatórios, onde o formato final dos mesmo resulta em um pdf. Como meio de persistência de dados convencionou-se usar o Database MySQL.
